World Disaster Informer
Web application that utilizes the Relief Web API and Google Maps API to track ongoing disasters across the world and display information about how the user can get involved in relief efforts.
Technologies Used: HTML, CSS, Bootstrap, Javascript, jQuery, AJAX, JSON
Community Art Curator
Web application that pulls from a MYSQL database to populate a map with locations, photos and descriptions of community art installations across the city of Atlanta. Also allows users to submit their own entries.
Technologies Used: HTML, CSS, Bootstrap, Javascript, jQuery, Express, MYSQL, Sequelize
Crystal Collector Game
Web game wherein each crystal is given a randomly generated value, and the player is asked to use the crystals to reach the given goal value.
Technologies Used: HTML, CSS, Bootstrap, Javascript, jQuery
Star Wars Hangman

Web game that asks the user 10 Star Wars questions with a 1 minute timer, and scores them at the end.
Technologies Used: HTML, CSS, Bootstrap, Javascript, jQuery
